split-name

(split-name {:keys [name]})
CircleCI combines the owner and project name into the cctray xml name attribute.

This function splits the name into individual entries in the project map.

So instead of:

    {:name "Owner/Project Name"}

You end up with:

    {:name   "Project Name"
     :owner  "Owner"}
